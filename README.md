# Language Learning Flashcards

A simple flashcard web application for checking English word translations.

## Live Demo

https://alexandrabalinskaiya.github.io/language-learning-flashcards/

## Description

Language Learning Flashcards is a small frontend practice project built with HTML, CSS, JavaScript and jQuery.

The application shows random English words, and the user needs to enter the correct Ukrainian translation. The app tracks correct and incorrect answers, shows the current step, and displays the user's language level after completing the test.

## Features

* random word selection
* flashcard interface
* translation input validation
* correct and incorrect answer counter
* step counter
* final result modal
* automatic game reset
* simple responsive layout

## Technologies

* HTML5
* CSS3
* JavaScript
* jQuery

## How to Use

1. Open the live demo.
2. Look at the English word on the flashcard.
3. Enter the Ukrainian translation.
4. Click the Check button.
5. Complete 10 steps.
6. The application will show your result level.
